# Human Activity Recognition Using GRU and Autoencoder Techniques
Overview
This project focuses on Human Activity Recognition (HAR) using a GRU-based Denoising Autoencoder to extract robust latent features from time-series sensor data. The extracted features are then classified using a Random Forest model to achieve high-accuracy activity recognition.

Technologies Used
Programming Language: Python
Libraries & Frameworks: TensorFlow/Keras, Scikit-learn
Models: GRU, Autoencoders, Random Forest

Features
Developed a GRU-based Denoising Autoencoder to extract meaningful features from sensor data.
Implemented noise reduction techniques to enhance model robustness.
Created data preprocessing pipelines, including normalization and Gaussian noise addition.
Applied Random Forest classification on extracted features, achieving high accuracy in HAR
