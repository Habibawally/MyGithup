Your ultimate travel guide to exploring **Alexandria, Egypt**! 🚀✨

---

## 📌 Overview

The **Travel Assistant App** is a Flutter-based mobile application designed to provide users with a seamless travel experience in Alexandria. It offers itinerary planning, real-time data updates, and a curated list of must-visit destinations.

---

## 🚀 Features

### 🗺️ Smart Itinerary Planner
- Users can plan and customize their travel itinerary.
- Predefined recommendations for **restaurants, hotels, attractions, and flights** in Alexandria.

### 🔐 Secure Authentication
- Firebase Authentication ensures a secure login and user management system.
- Supports Google and email-based authentication.

### 🔥 Real-time Data with Firebase
- Dynamic content updates with Firebase Firestore.
- Cloud storage for user preferences and travel history.

### ✈️ Flight Booking Wallet
- Users can book flights directly from the app.
- Integrated wallet system for secure and seamless transactions.
- Tracks past bookings and provides an optimized travel experience.

## 📲 Installation & Setup

1️⃣ **Clone the repository:**
```bash
 git clone https://github.com/Habibawally/travel-assistant-app.git
 cd travel-assistant-app
```

2️⃣ **Install dependencies:**
```bash
 flutter pub get
```

3️⃣ **Run the app:**
```bash
 flutter run
```

4️⃣ **Set up Firebase:**
- Create a Firebase project.
- Enable authentication (Google & Email sign-in).
- Configure Firestore database.
- Download and place `google-services.json` (Android) and `GoogleService-Info.plist` (iOS) in the correct directory.
